# 🤖 РезюмеAI

**AI-powered HR Assistant** — интеллектуальный анализ резюме и GitHub профилей с помощью локальной нейросети Ollama.

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://python.org)
[![FastAPI](https://img.shields.io/badge/FastAPI-0.100+-green.svg)](https://fastapi.tiangolo.com)
[![Ollama](https://img.shields.io/badge/Ollama-qwen3:8b-orange.svg)](https://ollama.ai)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## ✨ Возможности

- 📄 **Анализ PDF резюме** — извлечение текста и оценка качества
- 🐙 **Анализ GitHub профиля** — оценка активности, проектов и стека технологий
- 🎯 **ATS оптимизация** — рекомендации по ключевым словам для HR-систем
- 🎙️ **Голосовой отчёт** — озвучка результатов анализа (Edge TTS)
- 📊 **Детальная оценка** — отдельные баллы за резюме, GitHub и общую оценку
- 💡 **Конкретные советы** — actionable рекомендации для получения оффера
- 🔒 **Приватность** — всё работает локально, данные не уходят в облако

## 🚀 Быстрый старт

### Требования

- Python 3.10+
- Ollama (для работы AI)
- Node.js (опционально, для разработки)

### Установка

1. **Клонируй репозиторий:**
```bash
git clone https://github.com/yourusername/resume-ai.git
cd resume-ai
```

Этот проект доступен как **open-source** для личного использования и обучения.
